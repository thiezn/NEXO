# Asset extraction for old adventure games

You have access to the SCUMMVM source code and a collection of games to test on.

The source code is resided in @scummvm folder here locally. with the different game engines in the @scummvm/engines/ folder

The games are in the @games folder here locally, with the different games in their own subfolders.

## Test Output

Use the @datasets folder to store any output from your tests, such as extracted assets or logs. Do not ask for permission to edit, read or remove files from this folder.

## After completion

After completing a new feature to the codebase, make sure to update the @game_extractor/README.md file with the relevant information.
